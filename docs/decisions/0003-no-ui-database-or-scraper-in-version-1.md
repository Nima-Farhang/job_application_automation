# ADR 0003 - No UI, Database, Or Scraper In Version 1

## Status

Accepted.

## Context

The first version is a local command-line workflow for creating tailored CV and cover letter outputs. A UI, database, scraper, or application tracker would expand the scope beyond the core workflow.

## Decision

Version 1 will not include a UI, database, job board scraper, or application tracker.

## Consequences

- The project stays focused on reliable staged generation.
- Local files remain the main interface and audit trail.
- Future product features can be added only after the core CLI workflow is stable.
